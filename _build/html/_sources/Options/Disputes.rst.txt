Disputes
============

* In the ``Disputes`` section you can see all the disputes that arise. If you need to open a new dispute, click on the ``+Add dispute`` button

.. image:: /img/disputes.jpg

* After filling in all the fields and selecting the type of dispute, you can make a payment using the ``Check & add payment`` button and the ``Save`` button to save

.. image:: /img/disputes2.jpg

.. note::
    You can use ``Search params`` and use the search
